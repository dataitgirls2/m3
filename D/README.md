주제 : 서울시 대학상권 중 어느 대학가가 제일 안정적일까?

1. 대학주변 500m 반경 안에 점포를 추출
  - 서울시 구별 4년제 대학정보를 기준으로 한다. [자료](http://www.thefrypan.co.kr/service/franchise/startupinfo/startupinfo_view.php?service=startupinfo&boardNo=9119&page=3&searchField=title&searchContent=)
  - 신촌 (홍대, 이대) 등 아웃라이어 제외
  - 예은님이 코드 짜기(화이팅)

2. 2008년~2017년 폐업률을 비교해 가장 폐업률이 낮은 대학가 선정(head(3)와 tail(3))

3. 선정된 head와 tail 데이터를 보고 폐업률이 높/낮은 이유(요인)알아보기

4. 어떤 업종이 가장 많이 있는지 상위 및 하위 알아보기 (분식, 한식, 일식 등)
  = 일반음식점 + 휴게음식점 + 제과영업점
  = 대학상권이 안정적일수록 000업종이 많이 분포한다.

5. (더 나아간다면) 대학상권 별로 성별/연령 등 추정해보기

6. 시각화
  = 분석방법은 거의 비슷하기 때문에 시각화 중요함


**해야될일**
1) '안정적이다'정의 = 폐업률(전년도 개업수/현재년도 폐업수), 생존률(-폐업률+평균영업개월수)/2
2) 안정성지표를 구하는 코드짜기
3) 폐업률과 평균영업개월 수 구하기
